# shoping-system

shoping system

## Build Setup

``` bash
运行项目
1、先安装mongodb和环境搭建： http://www.cnblogs.com/ccyinghua/p/7887713.html

2、安装mongovue,建立dumall数据库，增加goods和users集合，插入数据(数据在resource/dumall-goods和resource/dumall-users)

3、npm install

4、node server/bin/www // 启动express后端服务

5、npm run dev
```
